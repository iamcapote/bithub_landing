Installation

BITCORE can be deployed as a Docker container or accessed via our managed cloud API.

BASH
npm install @bitcore/sdk
# or
pip install bitcore-py

Quick Start

Initialize a new agent swarm and connect it to your knowledge base.

JAVASCRIPT
import { BitCore } from '@bitcore/sdk';

const client = new BitCore({
  apiKey: process.env.BITCORE_KEY
});

// Create a reasoning loop
const swarm = await client.swarms.create({
  name: "ResearchAssistant",
  model: "llama-3-70b",
  tools: ["web_search", "vector_db"]
});

const response = await swarm.ask("Analyze the latest DeFi trends");
console.log(response.insight);

Nodes & Edges

Unlike traditional RAG, BITCORE structures data as a dynamic graph. Nodes represent entities (concepts, people, code), and Edges represent semantic relationships.
